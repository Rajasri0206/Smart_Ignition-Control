# Smart_Ignition-Control

## Description
This project is a vehicle security system using:
- RFID
- Fingerprint sensor
- GSM module
- Arduino

It provides secure access using dual authentication.

## Features
- RFID card authentication
- Fingerprint verification
- SMS alerts for unauthorized access
- Ignition control using relay

## Components Used
- Arduino
- RFID Module
- Fingerprint Sensor
- GSM Module
- LCD Display
- Relay
- Buzzer

## How it works
1. User scans RFID card
2. If valid → asks for fingerprint
3. If both match → ignition ON
4. If failed → SMS alert sent
